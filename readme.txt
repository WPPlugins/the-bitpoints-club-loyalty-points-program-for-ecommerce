== Installation ==

1. Upload "BitPointsClub.zip" to the "/wp-content/plugins/" directory
or
Use the Wordpress Plugin search to install from within Wordpress
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Sign up with The BitPoints Club. This will give you access to the Loyalty Points Engine and API Key.
1. Sign in to The BitPoints Club and complete the setup wizard. Remeber to choose The Points System, as this is the only option configured to work with Wordpress.
1. In Wordpress, go to the BitPoints menu option to configure.
1. Complete the fields as requested.
1. Save your changes and your loyalty program will be applied to your eCommerce store!
